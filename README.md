# News Crawler



## Objectives

지정된 검색어로 뉴스 수집 후 Dropbox에 업로드하는 프로그램



## Execute

1. Clone repository

```
git clone https://github.com/respect5716/News_Crawler.git
cd News_Crawler
```



2. Make .env file

```
vim .env

TIME=19:00 # time to crawl data
DROPBOX_TOKEN=abcde # dropbox access token
CONFIG_DIR=/app/config
```
